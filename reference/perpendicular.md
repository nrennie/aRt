# Perpendicular

This function generates a generative art ggplot object featuring
multiple coloured perpendicular lines.

## Usage

``` r
perpendicular(
  n = 100,
  max_length = 7,
  linewidth = 0.5,
  main_col = "black",
  bg_col = "white",
  s = 123
)
```

## Arguments

- n:

  Number of rectangles. Default 100.

- max_length:

  Maximum length of line. Default 7.

- linewidth:

  Line width of line. Default 0.5.

- main_col:

  Colour of lines. Default "black".

- bg_col:

  Background colour. Default "white".

- s:

  Seed value. Default 123.

## Value

A ggplot object

## Examples

``` r
perpendicular()
```
