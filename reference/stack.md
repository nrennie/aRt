# Stack

This function generates an sf object.

## Usage

``` r
stack(x0, y0, min_height, max_height, min_width, max_width)
```

## Arguments

- x0:

  x-coordinate of middle of rectangle.

- y0:

  y-coordinate of middle of rectangle.

- min_height:

  minimum height.

- max_height:

  maximum height.

- min_width:

  minimum width.

- max_width:

  maximum width.

## Value

An sf object.
